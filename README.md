# won1000
practice of Whole-Slide Imaging (WSI)


Requirements
============

* Python 2 >= 2.6 or Python 3 >= 3.3
* OpenSlide >= 3.4.0
* Python Imaging Library or Pillow


Installation
============

1.  `Install OpenSlide`_.

2.  ``pip install openslide-python``

.. _`Install OpenSlide`: https://openslide.org/download/
